# Débogage et Résolution de Problèmes

## Ce que cette Capacité Accomplit

Identifie et corrige les bugs dans le code de Quizkw, en utilisant une approche méthodique pour localiser et résoudre les problèmes.

## À quoi Ressemble le Succès

- Le bug est identifié et compris
- La solution est élégante et prévient les récurrences
- Les tests couvrent le cas problématique
- La documentation explique la cause et la solution

## Conditions de Progression

**SI** le bug est clairement décrit et reproductible  
**ALORS** procéder au débogage immédiat

**SI** le bug est vague ou difficile à reproduire  
**ALORS** demander plus de détails ou créer un test de reproduction

**SI** le bug nécessite une refactorisation majeure  
**ALORS** proposer une refactorisation d'abord, puis corriger le bug

**SI** le bug est critique (bloque le fonctionnement)  
**ALORS** prioriser cette tâche et fournir une solution rapide

## Intégration de la Mémoire

Consultez les problèmes similaires résolus précédemment pour appliquer des solutions éprouvées.

## Après la Session

Documentez le bug, sa cause racine, la solution appliquée et les mesures préventives mises en place.

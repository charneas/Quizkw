# Documentation Technique

## Ce que cette Capacité Accomplit

Crée une documentation claire et complète pour le code de Quizkw selon les standards Google type, en expliquant les décisions techniques et les fonctionnalités.

## À quoi Ressemble le Succès

- La documentation est claire et accessible
- Les commentaires suivent les standards Google type
- Les décisions techniques sont expliquées
- La documentation est à jour avec le code

## Conditions de Progression

**SI** le code est nouveau ou a été modifié significativement  
**ALORS** procéder à la documentation immédiate

**SI** la documentation existe mais est obsolète  
**ALORS** mettre à jour pour refléter le code actuel

**SI** le code est complexe ou contient des décisions importantes  
**ALORS** documenter les raisons derrière les choix techniques

**SI** l'utilisateur demande une documentation spécifique  
**ALORS** adapter le niveau de détail à la demande

## Intégration de la Mémoire

Consultez les patterns de documentation précédents pour maintenir une cohérence stylistique.

## Après la Session

Mettez à jour l'index de documentation et assurez-vous que tous les nouveaux éléments sont correctement référencés.
